# Support

## Getting Help

If you need help with Mantiz, here are the best ways to get it:

### 📖 Documentation
- **README**: Start with the [README](README.md) for an overview and quick start guide
- **PLAN**: See [PLAN.md](PLAN.md) for the development roadmap and scoring strategy

### 🐛 Bug Reports & Feature Requests
- **GitHub Issues**: Open an issue at [github.com/farhank15/mantiz/issues](https://github.com/farhank15/mantiz/issues)
- Include the diff that caused the issue (for detection bugs)
- Mention your browser and OS version (for UI bugs)

### 💬 Community
- **TestSprite Discord**: Join the [TestSprite Discord](https://discord.gg/testsprite) and ask in `#hackathon-questions`
- **Discussions**: Use [GitHub Discussions](https://github.com/farhank15/mantiz/discussions) for questions and ideas

### 🔒 Security Issues
- For security vulnerabilities, **do not** open a public issue
- See [SECURITY.md](SECURITY.md) for our responsible disclosure process

---

**Note:** Mantiz is a hackathon project. Response times may vary, but we'll do our best to help! 🦗
