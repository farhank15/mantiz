Integration Guide

GitHub Actions

Add this step to your workflow:

- name: Mantiz Scan
  uses: farhank15/mantiz@main
  with:
    api-token: ${{ secrets.MANTIZ_API_TOKEN }}
    threshold: 51

CLI

Run with your token:

npx @farhank15/mantiz-cli --token your_token_here --save

Use --save to persist results to your history.
